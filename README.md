## Utilisation
### Python
CSV dans ./data/
Le programme va récupérer autant de fonctions z=f(x) qu'il y a de CSV.
Il va ensuite faire une moyenne glissante et faire sauter des points (mean_length et step).
Puis il va 'empiler' sur y les fonctions récupérées pour obtenir une fonction z = f(x,y)
Finalement il sauve les données dans depth.dat et affiche quelques graphiques
